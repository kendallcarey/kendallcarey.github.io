Regular Expressions

Or Regex, which you have probably heard them referred to as, is a special text string describing a search pattern. This brief blog is not meant to teach you how to use regex efficiently but hopefully can be used as a quick reference of Regex:

  • [abc]
  • A single character of: a, b, or c
  • [^abc]
  • Any single character except: a, b, or c
  • [a-z]
  • Any single character in the range a-z
  • [a-zA-Z]
  • Any single character in the range a-z or A-Z
  • ^
  • Start of line
  • $
  • End of line
  • \A
  • Start of string
  • \z
  • End of string
  • .
  • Any single character except line breaks
  • \s
  • Any whitespace character
  • \S
  • Any non-whitespace character
  • \d
  • Any digit
  • \D
  • Any non-digit
  • \w
  • Any word character (letter, number, underscore)
  • \W
  • Any non-word character
  • \b
  • Any word boundary
  • (...)
  • Capture everything enclosed
  • (a|b)
  • a or b
  • a?
  • Zero or one of a
  • a*
  • Zero or more of a
  • a+
  • One or more of a
  • a{3}
  • Exactly 3 of a
  • a{3,}
  • 3 or more of a
  • a{3,6}
  • Between 3 and 6 of a
  • g
  • global
  • [s/S/]
  • matches any character including line breaks
  • %
  • string
  • options: i case insensitive, m make dot match newlines, x ignore whitespace in regex, o perform #{...} substitutions only once
  • all escaped characters start with \

    • So breaking down \b[A-Z0-9._%+-]+@[A-Z0-9.-]+\.[A-Z]{2,4}\b uses the any word boundaries, any character a to z and 0 to 9 one or more characters adds it to a string, adds the @, takes any character a to z, 0 to 9 then adds a ‘.’ and uses between 2 and 4 characters between a to z. This searches for an email address. Any email address. Pretty cool right? Fun stuff.

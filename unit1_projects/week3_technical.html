Why is JavaScript so popular?

JavaScript is extremely popular for many reasons. Having spoken to many programmers, each of them stated that they believed that JavaScript is the future of programming and that everyone should learn it. At the very beginning, JavaScript was the language of Netscape. If you wanted to do scripting on the Internet back then, JavaScript was it. We continue to use it not only because it’s easy to learn, easy to edit and debug, and its quick and responsive but all browsers support it, it can be used on the client as well as the server side, and it’s a universal front end platform. JavaScript is everywhere and for good reason. I think the main reason is its growing in popularity is because multiple devices have JavaScript. Everything from tablets to smart phones supports it. A new devices grow in popularity, more jobs are created for web developers that know JavaScript.
